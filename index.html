autotwin is an automated workflow used for segmentation, meshing, and analysis of a digital twin.

The workflow is useful for scientific analysis of as-built and in situ domains, where the geometric description of the subject comes from image stacks.

Segmentation

ensemble is an automated Python workflow that creates a segmentation from image stacks.

Meshing

automesh is an automated Rust workflow that creates an all-hexahedral finite element mesh from a segmentation.

Analysis

autosim is an automated Python workflow that creates analysis data sets from a finite element solver, such as ABAQUS or Sierra Solid Mechanics.
